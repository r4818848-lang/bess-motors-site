# Сайт без стилей (белый фон, синие ссылки) — исправление

## Причина
Не загружается **Tailwind CSS**. Синяя «машина» — это логотип внутри ссылки (цвет ссылки = синий).

## Решение (выполните по порядку)

### 1. Остановите сервер
В терминале где запущен сайт: **Ctrl + C**

### 2. Перейдите в правильную папку
```powershell
cd c:\Users\Asus\Desktop\website\bess-motors
```
**НЕ** папка `website` (старый шаблон) — только **bess-motors**!

### 3. Очистите кэш и переустановите
```powershell
Remove-Item -Recurse -Force .next -ErrorAction SilentlyContinue
npm.cmd install
```

### 4. Запустите снова
```powershell
npm.cmd run dev
```

### 5. Откройте в браузере
http://localhost:3000

**Жёсткое обновление:** Ctrl + Shift + R (очистить кэш страницы)

---

## Как должно выглядеть
- Чёрный фон
- Белый текст, красные акценты
- Нормальный header, не огромная синяя машина

## Если всё ещё белый экран
Проверьте в браузере F12 → вкладка **Network** → обновите страницу → найдите файл `layout.css` или `[hash].css` — статус должен быть **200**, не 404.

Напишите, что показывает терминал после `npm.cmd run dev`.
